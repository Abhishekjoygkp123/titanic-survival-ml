# Titanic Survival Prediction

A machine learning project built as preparation for a Data & AI internship.

## What this project does
- Loads and cleans the Titanic dataset
- Selects relevant features for survival prediction
- Builds a Decision Tree classifier using scikit-learn
- Improves accuracy from 65.9% to 71.5% by fixing overfitting with max_depth=5
- Uses Groq AI API to explain the model results in plain English

## Tools used
- Python, pandas, scikit-learn, matplotlib
- Groq API (LLaMA 3.1)
- Google Colab, GitHub

## Model Accuracy
71.5% on test data
